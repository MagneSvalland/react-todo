# react-todo

🔗 **[Live demo](https://react-todo-gold-psi.vercel.app)**

A lightweight task manager built with React, TypeScript and Vite.

## Features

- Add tasks with **priority levels** (low / medium / high)
- **Due dates** with overdue warning
- **Inline editing** — double-click any task to edit
- **Filter** by all / active / completed
- **Dark/light mode** toggle
- **localStorage** persistence — tasks survive page refreshes
- Clear all completed tasks at once

## Tech stack

| Layer | Tech |
|---|---|
| Framework | React 18 |
| Language | TypeScript |
| Bundler | Vite |
| Styling | Vanilla CSS (custom properties) |
| State | useState + custom hooks |
| Persistence | localStorage |
